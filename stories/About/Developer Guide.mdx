<h3 id = "started">Getting Started</h3>
<ol>
  <li>Clone the develop branch to your local machine:</li>
  <li><code>git clone --single-branch --branch develop https://github.com/LetsGetTechnical/gridiron-survivor.git</code></li>
  <li>Install the required packages with pnpm (NOT npm or yarn!)</li>
  <li><code>pnpm install</code></li>
  <li>Create a new branch for whatever you‚Äôre working on in the format firstname/feature-description, then check out your new branch. For example:</li>
  <li><code>git branch richard/user-picks-page && git checkout richard/user-picks-page</code></li>
  <li>Code your feature! ü§ì</li>
</ol>

<h3 id = "push">Pushing local branch to remote branch</h3>
<ol>
<li>For the first push from local to remote, use --set-upstream origin along with the name of your local branch, like so:</li>
<li><code>git push --set-upstream origin richard/user-picks-page</code></li>
<li>Subsequent pushes will now go to your remote upstream branch.</li>
</ol>
